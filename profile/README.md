# Aqta

**Trust infrastructure for AI in regulated industries.**

We build the cryptographic evidence layer underneath AI in finance, healthcare, and the public sector. Every AI request routed through AqtaCore returns a signed receipt. Every action is provable. Every action is auditable. No vendor-controlled log to dispute.

## Public artefacts

🔐 **[attestation-spec](https://github.com/Aqta-ai/attestation-spec)** — Open specification for AqtaCore receipts. Two reference verifier libraries (Python, TypeScript). Conformance test suite. CC BY 4.0 spec, Apache 2.0 code.

📦 **[aqta-verify-receipt (PyPI)](https://pypi.org/project/aqta-verify-receipt/)** — `pip install aqta-verify-receipt`. Verify any AqtaCore receipt without trusting Aqta's servers.

📦 **[aqta-verify-receipt (npm)](https://www.npmjs.com/package/aqta-verify-receipt)** — `npm install aqta-verify-receipt`. TypeScript verifier for the same spec.

🌍 **[aqtabio-research](https://github.com/Aqta-ai/aqtabio-research)** — Methodology, governance framework, MCP server source, regulatory pathway.

✍️ **[bounds](https://github.com/Aqta-ai/bounds)** — Browser-only PDF redaction (free tier).
