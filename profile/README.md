# Aqta

**Trust infrastructure for AI in regulated industries.**

Every AI decision in finance, healthcare, and the public sector returns a signed receipt: provable, auditable, and verifiable by anyone against one public key, offline, without trusting us.

Aqta Technologies Limited · Dublin, Ireland

## Live

- **🛡️ [AqtaCore](https://app.aqta.ai)** — runtime policy enforcement with Ed25519-signed attestation receipts. Audit-ready by design, not by report.
- **🌍 [AqtaBio](https://aqtabio.org)** — public-health AI that commits spillover predictions to a public ledger before events occur. Our first prospective Ebola signal preceded the WHO PHEIC by eight days. [Methodology and ledger](https://github.com/Aqta-ai/aqtabio-research).

## Open

- **[attestation-spec](https://github.com/Aqta-ai/attestation-spec)** — the open spec for AqtaCore receipts, with reference verifiers and a conformance suite. CC BY 4.0 spec, Apache 2.0 code.
- **aqta-verify-receipt** — verify any receipt without trusting our servers. [`pip install aqta-verify-receipt`](https://pypi.org/project/aqta-verify-receipt/) · [`npm i aqta-verify-receipt`](https://www.npmjs.com/package/aqta-verify-receipt)
- **[bounds](https://github.com/Aqta-ai/bounds)** — browser-only PDF redaction.
