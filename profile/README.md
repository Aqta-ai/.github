# Aqta

**The trust layer for AI in regulated industries.** Every governed AI call returns a signed receipt: verifiable by anyone against one public key, offline, without trusting us.

## Live

- **[AqtaCore](https://app.aqta.ai)** — runtime policy enforcement with Ed25519-signed receipts. Audit-ready by design, not by report.
- **[AqtaBio](https://aqtabio.org)** — public-health AI that commits spillover predictions to a public ledger before the event. Our first Ebola signal was on the ledger eight days before the WHO declared the emergency. [Methodology and ledger](https://github.com/Aqta-ai/aqtabio-research).

## Open

- **[attestation-spec](https://github.com/Aqta-ai/attestation-spec)** — the open spec for AqtaCore receipts, with reference verifiers and a conformance suite. CC BY 4.0 spec, Apache 2.0 code.
- **[aqta-verify-receipt](https://pypi.org/project/aqta-verify-receipt/)** — verify any receipt without our servers. `pip install aqta-verify-receipt` · `npm i aqta-verify-receipt`
- **[bounds](https://github.com/Aqta-ai/bounds)** — browser-only PDF redaction.

Everything else is proprietary.
